# CapyQuest 3: Mask Matrix
CapyQuest 3 is a puzzle maze game. Development originally began during the Global Game Jam 2026, where the theme was “Masks,” and the project later continued as part of the CapyQuest series.

Players control Jeff as he navigates through a digital maze in cyberspace while collecting special items hidden throughout the network. The core mechanic revolves around color masks: activating a mask disables obstacles of that color, allowing Jeff to pass through blocked areas. However, switching masks can also block access to keys or paths the player still needs, so the challenge comes from carefully deciding when to toggle each color in order to solve the maze and reach the goal.
